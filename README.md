## Chat2VK

Возможности плагина:

Отправка сообщений с сервера в беседу/личку админа VK и наоборот

Проверка онлайна (карта, фраги игроков, steamid) (в т.ч. со всех серверов сразу) из VK

Исполнение rcon команд из VK
